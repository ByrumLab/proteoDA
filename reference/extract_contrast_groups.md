# Extract groups from a vector of contrasts

Extract groups from a vector of contrasts

## Usage

``` r
extract_contrast_groups(contrast.vec)
```

## Arguments

- contrast.vec:

  A vector of contrasts, most likely from reading in a contrasts text
  file

## Value

A named list of length equal to the number of contrasts. Each element in
the list is a character vector of the groups involved in that contrast,
and the names of the list give the contrast names.
