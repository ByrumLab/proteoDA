# Convert a column to a factor

Converts a vector/column to a factor. For numeric inputs, adds an "X" in
front of the number by default, or can supply your own prefix to use
instead.

## Usage

``` r
make_factor(x, prefix = "X")
```

## Arguments

- x:

  Input vector to be converted to a factor

- prefix:

  Optional: a prefix to put in front of numbers when converting to a
  factor. Default is "X"

## Value

A factor vector
