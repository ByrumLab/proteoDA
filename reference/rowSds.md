# Calculate per-row standard deviations of a numeric array

Have done testing, gives same results as the genefilter::rowSds()
function.

## Usage

``` r
rowSds(x, ...)
```

## Arguments

- x:

  The array for which to calculate per-row Sds

- ...:

  Additional arguments to be passed to internal functions. Meant for
  na.rm.

## Value

A numeric vector of appropriate length, named if input was named, with
per-row standard deviations
